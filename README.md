# Professional Portfolio (React + Node.js)

## Setup Instructions

### Backend (Node.js)
```bash
cd backend
npm install
npm start
```
Server chalega: http://localhost:5000

### Frontend (React)
```bash
cd frontend
npm install
npm start
```
App chalega: http://localhost:3000

## Features
✅ Hero Section with CTA buttons
✅ About Me section
✅ Skills (Tech Stack) categorized
✅ Projects showcase with GitHub & Live links
✅ Experience & Education timeline
✅ Contact form with email integration
✅ Dark/Light mode toggle
✅ Resume download button
✅ Fully responsive design

## Tech Stack
- **Frontend**: React, CSS3
- **Backend**: Node.js, Express, Nodemailer
- **API**: REST API

## Email Setup
Backend ke `server.js` mein apna Gmail aur App Password add karein:
```javascript
user: 'your-email@gmail.com',
pass: 'your-app-password'
```
